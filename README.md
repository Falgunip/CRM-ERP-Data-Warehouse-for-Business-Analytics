# CRM-ERP-Data-Warehouse-for-Business-Analytics
This project demonstrates the integration of CRM (Customer Relationship Management) and ERP (Enterprise Resource Planning) datasets into a centralized data warehouse.  The goal is to organize, structure, and model business data so it can be used for insights, reporting, and decision-making.

🎯Objectives
- Combine CRM and ERP data into a structured warehouse
- Build fact and dimension tables for analytics
- Enable efficient reporting and business insights
- Analyze customer behavior, sales, and operational trends
